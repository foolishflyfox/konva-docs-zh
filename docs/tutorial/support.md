# 需要关于 Konva 库的帮助？

> 原文地址: https://konvajs.org/docs/support.html

## 正在寻找关于 Konva 框架的帮助？

下面是你接下来应该做的事：

1. 实现尝试在线找到解决方案。尝试通过 google 或本页面顶部的搜索按钮寻找你问题的解决方案。
2. [StackOverflow](https://stackoverflow.com/questions/tagged/konvajs) 是最好的提问的地方。如果你提出的是一个带有在线示例、代码片段、正确标签等的高质量问题，你将有更大的机率收获一个满意的答案。
3. 如果你发现了一个 bug 或者你希望官方支持一个新功能，请去 [问题页面](https://github.com/konvajs/konva/issues)。
4. 如果你只是希望加入关于 Konva 的讨论，可以加入 [discard 交流](https://discord.com/invite/8FqZwVT)。
5. 如果你有一些关于 Konva 的有趣的内容想分享，可以使用 Twitter `#konvajs` 标签。
6. 访问 [修改日志](https://github.com/konvajs/konva/blob/master/CHANGELOG.md) 查看正在改变的内容。
7. 需要咨询或策略评估？请前往[咨询页面](https://lavrton.com/consulting/)。
