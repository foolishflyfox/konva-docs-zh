# 捐赠

> 原文地址: https://konvajs.org/docs/donate.html

你好，我叫 Anton。我是 Konva 框架的核心维护者。

**如果你希望支持 Konva 及其生态工具如 react-konva 和 vue-konva 的开发**，你可以使用：

- [Patreon 创作者平台](https://www.patreon.com/lavrton)
- [Open Collective 财务管理平台](https://opencollective.com/konva)
- [GitHub 捐赠者](https://github.com/sponsors/lavrton)

我将大量的时间用于支持 Konva 用户及开发新版本，包括 Bug 修复及新功能开发。

如果你正在项目中使用 Konva，并已经借此赚到了钱，那么支持 Konva 的开发将非常有意义。因为这样，你将确保你拿到的是一个高质量的，被维护的框架。

即使你没有从你的项目中挣到钱，但 Konva 帮你省了大量时间，那么支持它也很有意义。

## 你的公司正在使用 Konva？

让广大开发者自掏腰包进行捐款可能有些强人所难。但如果您所在的公司正在使用 Konva，不妨和领导聊聊，争取得到公司的支持。一个被良好维护的项目，对公司来说也不失为一笔很划算的投资。
