# Container 容器类

Konva.Container 是一个抽象类，表示一个容器，用于包含节点或其他容器，例如 Stage、Layer 及 Group 都继承自该类。

## 构造函数

构造函数参数为 config，包含如下字段：

| 字段名          | 类型     | 描述 |
| --------------- | -------- | ---- |
| `x` (可选)      | `Number` |      |
| `y` (可选)      | `Number` |      |
| `width` (可选)  | `Number` |      |
| `height` (可选) | `Number` |      |
